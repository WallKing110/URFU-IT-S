В строку "four bites" вводится двоичный код, над которым мы собираемся проводить преобразования. Он может быть не обязательно четырехзначный.
Кнопка "To code" добавляет в значение из "four bites" контрольные нули и единицы для дальнейших преобразований(см ход хэмминга) и выводит новое значение в "encoded message".
Строка "encoded message" показывает строку из "four bites" с контрольными битами после нажатия кнопки "To code". Также есть возможность ввести уже готовую строку(если у вас она есть).
Кнопка "To hurt 1 bite" заменяет один случайный символ из "encoded message" на противоположный по значению(0 --> 1, 1 --> 0) и выводит новую строку в "hurted message".
Строка "hurted message" показывает строку из "encoded message" с одним замененным символом после нажатия кнопки "To hurt message". Также есть возможность ввести уже готовую строку(если у вас она есть).
Кнопка "To decode" находит поврежденный бит и восстанавливет изначально введеное значение, выводя его в "decoded message".
Строка "decoded message" выводит преобразованну строку из "hurted message" после нажатия на кнопку "To decode".
