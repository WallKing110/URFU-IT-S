Запуск:
cmd -> node rle.js (param1) (param2) (param3)
param1 - Параметр, в котором указывается декодировать нужно или закодировать. Значения: encode (закодировать), любое другое будет декодировать.
param2 - Параметр, в котором указывается путь к файлу, с которого нужно считать строку для дальнейшей кодировки.
param3 - Параметр, в коором указывается пусть к файлу, в который нужно будет сохранить закодированную/раскодированную строку.
